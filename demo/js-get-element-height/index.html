渲染畫面

inner

說明

這篇要使用瀏覽器原生 API，取得上方 div 元素的高度，並比較不同 API 的差異。
  • 透明 - margin  - 20px
  • 金色 - border   - 20px
  • 深綠 - padding - 20px
  • 淺綠 - content  - 120px 可視範圍 - 400px 總高度

使用原生 API 取得的高度

    原始碼

    
    // html
    <div id="box">
      <div id="inner">
        <h2>inner</h2>
      </div>
    </div>
    
    
    // css
    #box {
      width: 200px;
      height: 200px;
      overflow: auto;
      margin: 20px;
      border: 20px solid;
      padding: 20px;

      border-color: darkgoldenrod;
      background: darkslategray;
    }

    #inner {
      width: 100%;
      height: 400px;
      background: darkolivegreen;
      color: #fff;
    }
  
    
    // JS
    function start() {
      var box = document.querySelector("#box");
      var clientH = box.clientHeight;
      var scrollH = box.scrollHeight;
      var offsetH = box.offsetHeight;

      document.querySelector("#output").innerHTML = `
        <li>el.clientHeight = ${clientH} = padding + content (visible) </li>
        <li>el.scrollHeight = ${scrollH} = border + padding + content (visible + invisible)</li>
        <li>el.offsetHeight = ${offsetH} = border + padding + content (visible) = border-box Height</li>
      `;
    }

    window.addEventListener("load", start);
