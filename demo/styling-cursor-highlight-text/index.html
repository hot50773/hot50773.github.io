使用 CSS 偽元素 ::selection，修改文字反白樣式

  • ::selection 只支援少數屬性的修改：例如顏色
  • ::selection 是在CSS3之後才有，所以在IE9+才支援
  • 在Firefox中需要加前缀“-moz”

::selection {
	background: 背景顏色值;
	color: 文字顏色值;
}
::-moz-selection {
	background: 背景顏色值;
	color: 文字顏色值;
}

允許的屬性

只有一小部分 CSS 屬性可以用於 ::selection 選擇器：
  • color
  • background-color
  • cursor
  • caret-color
  • outline and its longhands
  • text-decoration and its associated properties
  • text-emphasis-color (en-US)
  • text-shadow

要特别注意的是，background-image 会如同其他屬性一樣被忽略。
